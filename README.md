# 📈 UV-Vis Spectroscopic & Band Gap Analyzer

نظام ذكي ومستقل متكامل مخصص للتشخيص والتحليل الرقمي لبيانات أطياف الامتصاص المستخرجة من جهاز الـ **UV-Vis Spectrophotometer**. يتيح النظام للباحثين حساب فجوة الطاقة البصرية تلقائياً ومطابقتها مع المراجع القياسية العالمية لتعيين هوية ونقاوة المواد النانوية والأكاسيد الكيميائية.

---

## 🚀 الميزات الأساسية للمنظومة (Key Features)

* **الاشتقاق التلقائي لفجوة الطاقة البصرية:** يقوم النظام بقراءة ملفات الإكسل والـ CSV المعملية وحساب حافة الامتصاص الحرجة (Absorption Edge) بدقة فيزيائية عالية.
* **رسم مخطط تاوك القياسي (Automated Tauc Plot):** يدعم الحسابات الرياضية للانتقالات الإلكترونية المختلفة:
  * الانتقال المباشر المسموح (Direct Allowed Transition, $n=1/2$).
  * الانتقال غير المباشر المسموح (Indirect Allowed Transition, $n=2$).
* **المطابقة مع مراجع اللامدا ماكس ($\lambda_{max}$):** يحتوي النظام على قاعدة بيانات مرجعية مدمجة وموسعة تشمل **1000 مادة وتوصيف فيزيائي** (تأثير التشويب Doping، المعالجات الحرارية Annealing، والنقاط الكمومية Quantum Dots).
* **تحديد هوية العينة (Phase Identification):** يقوم النظام تلقائياً بمسح القمم المكتشفة ومقارنتها بالقيم المرجعية المحسوبة وفقاً للمعادلة الكهرومغناطيسية:
  $$\lambda_{max} \ (nm) = \frac{1240}{E_g \ (eV)}$$
* **واجهة مستخدم تفاعلية واحترافية:** مبنية بالكامل باستخدام `Streamlit` مع دعم الرسم البياني المتقدم عبر `Matplotlib`.
